About FlowerAI

FlowerAI is an advanced machine learning project designed to identify flower species using deep learning techniques. Our CNN model achieves 94.5% accuracy across 5 different flower categories using a comprehensive dataset from Kaggle.

This project demonstrates the practical application of computer vision and neural networks in botanical classification, making flower identification accessible to everyone.
